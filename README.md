##Qué hace tu proyecto.##

Lo que realiza mi proyecto es una galeria que al momento de hacer click en cualquier imagen abre un modal para poder hacer en grande la imagen seleccionada y en ella podremos cambiar para ver las demás en tamaño mayor
al que se encontraba

##Cómo se usa. ##

Utiliza codigo en JS que este es funciona gracias al addEventListener añadiendo la funcion de click.

##Tecnologías aplicadas (HTML5, CSS3, JS, Git/GitHub).##

Utilice html para realizar mi galeria de forma ordenada y seccionada, además utlice CSS3 para darle un diseño bastante basico pero ordenado para que puedan apreciar de buena forma las imagenes
en cada imagen agregue un hover para que vea que imagen es la que quieren seleccionar, y finalmente utilice js para hacerlo funcionar y que al momento de hacer click en cada una de las
imagenes puedan maximizarme en la pantalla y así apreciarlas de mejor forma.
